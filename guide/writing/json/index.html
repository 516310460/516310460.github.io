JSON

Nuxt Content can query JSON formatted content.

If the document root is an object {}, the output contains all the document properties at its root. If the document root is an array [], the output contains all the document properties in a body property.

Example

content/hello.json
{  "title": "Hello Content v2!",  "description": "The writing experience for Nuxt 3",  "category": "announcement"}
Output
{  _path: '/hello',  _draft: false,  _partial: false,  title: 'Hello Content v2!',  description: 'The writing experience for Nuxt 3',  category: 'announcement',  _id: 'content:hello.json',  _type: 'json',  _source: 'content',  _file: 'hello.json',  _extension: 'json'}
